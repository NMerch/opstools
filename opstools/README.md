Usage:
---------
Provisions and optionally bootstraps a given number of EC2 instances.
If node name (-n) and -b are not used instances will be provisioned but not bootstrapped.

'AWS_KEY_ID', 
'AWS_SECRET_KEY', 
'REGION', 
'base_security_group',
'default_ssh_key'
